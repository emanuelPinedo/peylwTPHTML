INTRODUCCIÓN HTML

Este es un curso que pretende introducir al lector en el mundo del
HTML, No se requieren conocimientos previos, se emepzará desde cero
No hay que saber nada de programación para utilizar HTML, basta
aprender un par de cosas sencillas para emepzar a crear páginas.

Es MUY FÁCIL aprender HTML Tan sólo necesitas un editor de texto, un
curso de html y un navegador web (como netscape o internet explorer),
ni siquiera necesitas estar conectado a Internet.
